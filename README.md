# Vroha 2.0 - Business Website

A modern business website built with Next.js and deployed on Vercel.

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone this repository:
```bash
git clone https://github.com/Asyed9565/Vroha-2.0.git
cd Vroha-2.0
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view your site.

## Deployment on Vercel

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Click "New Project"
4. Select your GitHub repository
5. Click "Deploy"

That's it! Your site will be live on Vercel.

## Customization

Edit the files in the `app` and `public` folders to customize:
- Text content
- Colors and styling
- Images
- Layout
